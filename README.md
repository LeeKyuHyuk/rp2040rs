# rp2040rs

A Raspberry Pi Pico Emulator (WiP). It (almost) blinks!

Write [rp2040js](https://github.com/wokwi/rp2040js) project in **Rust**.

## Enjoy the wonders

```sh
git clone https://github.com/LeeKyuHyuk/rp2040rs.git
cd rp2040rs
cargo build
./target/debug/rp2040rs
```
